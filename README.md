# iterm-keys

First, go to Preferences -> Profiles -> Keys.

### Delete word

Send Hex Code: `0x1B 0x08`

### Delete line

Send Hex Code: `0x15`

### Navigate 1 word back

Send Escape Sequence: `b`

### Navigate 1 word in front

Send Escape Sequence: `f`
